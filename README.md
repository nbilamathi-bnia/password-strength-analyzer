# Password Strength Analyzer

## Overview

Password Strength Analyzer is a web-based application that evaluates the strength of user-entered passwords based on multiple security criteria. The tool provides real-time feedback, identifies weak passwords, and suggests improvements to help users create stronger and more secure passwords.

## Features

* Real-time password strength analysis
* Password strength meter (Weak, Medium, Strong)
* Password visibility toggle (Show/Hide Password)
* Common password detection
* Secure password generator
* Password improvement suggestions
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Evaluation Criteria

The application analyzes passwords based on:

1. Minimum length requirement (8+ characters)
2. Uppercase letters (A-Z)
3. Lowercase letters (a-z)
4. Numeric characters (0-9)
5. Special characters (!@#$%^&*)

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using any modern web browser.
4. Enter a password to analyze its strength.
5. Use the password generator to create a secure password.
